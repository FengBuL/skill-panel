# 现行开发文档

## 必读顺序

1. 仓库根目录 `PROJECT_STATE.md`：确认当前版本、正式版本和阻塞项。
2. 仓库根目录 `CURRENT-PLAN.md`：确认当前唯一任务和执行状态。
3. `AGENTS.md`：确认分支、原型、测试、数据保护和发布规则。
4. [产品需求 PRD](./product/PRD.md)：确认产品范围和信息架构。
5. [UI 规范](./product/UI-SPECIFICATION.md)：确认页面结构与交互验收。
6. [当前架构](./architecture.md)：确认源码入口和模块边界。
